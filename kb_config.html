1️⃣ DeepSeek API（必填）

用于 Arkitect AI 对话和 OCR 分析。在 platform.deepseek.com 获取。

2️⃣ 本地 OCR 检测器

如果你要在本地运行 OCR 脚本（MuMu模拟器截图+AI分析），请填写以下参数。只用网页版可跳过。

3️⃣ 下载本地工具

把以下文件下载到同一个项目文件夹中，然后在该文件夹中双击 setup.bat 即可一键启动。
使用步骤：
1. 填好 API Key 和 ADB 路径 → 点下方 💾 保存 → 点 🐍 导出给Python → 得到 _api_config.json
2. 创建项目文件夹，把下载的文件 + _api_config.json 全放进去
3. 把 spacerpg4_map_init.json 重命名为 spacerpg4_map_data.json
4. 先运行 🔌连接测试确认 ADB 能连上 MuMu
5. 双击 setup.bat → 选择 1 或 2
6. 同时打开星图网页，OCR 结果会自动同步
